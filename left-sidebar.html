Covid Dashboard

This dashboard features highly interactive and user friendly elements to help facilitate the ease of data retrieval. The dashboard comes complete with estimated cases if the pandemic were to continue, aswell as total death confirmed cases and deaths from the virus.

Dashboard Features

Due to the dashboard being created on tableau, it features highly interactive information. The Data in the dashboard were split up into the following categories:

  • Percent Population Infected per Country.
  • Percent Population Infected.
  • Total Deaths per Continent.
  • Total Deaths.
  • Total Cases.

You can highlight, or press on any aspect of the dashboard to get precise information regarding the data. You can also directly retrieve data for the desired aspect.
